# Moovie Extension

Browser extension for [moovie.fun](https://moovie.fun) — smooth direct CDN playback.

## Downloads

See [Releases](https://github.com/iMayhem/Moovie-Extension/releases) for browser-specific **ZIP** packages:

| Browser | File |
|---------|------|
| Chrome | `moovie-chrome.zip` |
| Edge | `moovie-edge.zip` |
| Firefox | `moovie-firefox.zip` |
| Opera | `moovie-opera.zip` |
| Brave / Vivaldi | `moovie-brave.zip` / `moovie-vivaldi.zip` |

Install via your browser store, **Load unpacked** (developer mode), or temporary add-on loading in Firefox.

Source lives in [iMayhem/watchable](https://github.com/iMayhem/watchable) under `extension/`.
