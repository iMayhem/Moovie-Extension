# Moovie Extension

Browser extension for [moovie.fun](https://moovie.fun) — smooth direct CDN playback.

## Downloads

See [Releases](https://github.com/iMayhem/Moovie-Extension/releases) for browser-specific packages:

| Browser | File |
|---------|------|
| Chrome | `moovie-chrome.zip` or `moovie.crx` |
| Edge | `moovie-edge.zip` |
| Firefox | `moovie-firefox.zip` or `moovie.xpi` |
| Opera | `moovie-opera.zip` |
| Brave / Vivaldi | `moovie-brave.zip` / `moovie-vivaldi.zip` |

Source lives in [iMayhem/watchable](https://github.com/iMayhem/watchable) under `extension/`.
